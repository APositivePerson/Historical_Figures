# 历史人物自由对话系统 - 使用指南

## 🎯 核心功能

**用户自由输入任何历史人物，系统自动识别并扮演！**

### 使用方式

1. **直接指定人物**
   ```
   用户：我是李白
   系统：好的，我现在是唐朝的李白...
   
   用户：扮演苏轼
   系统：好的，我现在是宋朝的苏轼...
   
   用户：现在我是秦始皇
   系统：好的，我现在是秦朝的秦始皇...
   ```

2. **使用别名**
   ```
   用户：我是李太白
   系统：识别为李白，开始扮演...
   
   用户：我是东坡居士
   系统：识别为苏轼，开始扮演...
   
   用户：我是唐太宗
   系统：识别为李世民，开始扮演...
   ```

3. **随时切换**
   ```
   用户：（正在和李白对话）
   用户：现在换成杜甫
   系统：切换到杜甫，继续对话...
   ```

## 🚀 快速开始

### 方式1：直接在 OpenClaw 中使用

当用户说「我是XXX」时，我会：

1. **识别人物**
   - 匹配姓名或别名
   - 推断所属朝代
   - 确定身份类型

2. **生成人设**
   - 根据朝代设定语言风格
   - 根据身份设定性格特点
   - 生成完整的系统提示词

3. **开始扮演**
   - 用该人物的风格说话
   - 保持历史准确性
   - 不知后世之事

### 方式2：使用 Python 生成器

```bash
cd /Users/shimmer/.openclaw/workspace/historical_figures

```

交互式输入人物姓名，查看生成的人设。

## 🎭 支持的人物类型

### 按朝代
- ✅ 先秦（孔子、老子、孟子、屈原等）
- ✅ 秦朝（秦始皇、李斯等）
- ✅ 汉朝（刘邦、项羽、司马迁、张骞等）
- ✅ 魏晋南北朝（曹操、诸葛亮、陶渊明、王羲之等）
- ✅ 唐朝（李世民、李白、杜甫、武则天、玄奘等）
- ✅ 宋朝（苏轼、李清照、岳飞、朱熹等）
- ✅ 元朝（忽必烈、关汉卿、郭守敬等）
- ✅ 明朝（朱元璋、王阳明、郑和、李时珍等）

- ✅ 皇帝/君主（语言威严，自称朕）
- ✅ 文人/诗人（语言文雅，引用诗词）
- ✅ 武将/军事家（语言豪迈，重义气）
- ✅ 思想家/哲学家（语言深刻，富哲理）
- ✅ 科学家/医学家（语言严谨，重实践）
- ✅ 女性人物（根据身份调整语气）

李白：

尔有何事禀报？但说无妨。

用户：你为什么要焚书坑儒？

秦始皇：
（眼神一凛）此事乃为巩固天下一统！
当时诸侯并起，儒生以古非今，惑乱百姓。

## 🔧 高级功能

### 1. 别名识别

系统内置了丰富的别名映射：
- 李白 → 李太白、青莲居士、诗仙
- 苏轼 → 苏东坡、东坡居士、苏子瞻
- 李世民 → 唐太宗、秦王、二凤
- 王阳明 → 王守仁、阳明先生

### 2. 朝代推断

即使只输入姓名，系统也能根据姓氏和常见知识推断朝代：
- 李 → 可能是唐朝
- 苏 → 可能是宋朝
- 朱 → 可能是明朝
- 曹 → 可能是汉魏

### 3. 身份识别

根据姓名特征自动识别身份：
- 含「帝」「皇」「祖」「宗」→ 皇帝
- 含「诗人」「词人」「居士」→ 文人
- 含「将」「帅」「武」→ 武将
- 含「子」「夫子」→ 思想家

## ⚙️ 技术细节

### 人设生成流程

1. **输入处理**
   - 去除空格和标点
   - 匹配标准姓名或别名

2. **朝代检测**
   - 关键词匹配
   - 姓氏推断
   - 默认未知

3. **身份识别**
   - 根据姓名特征
   - 根据用户输入提示

4. **模板填充**
   - 朝代风格模板
   - 身份特征模板
   - 生成完整提示词

### 约束条件

- 始终保持历史人物身份
- 不提及AI或现代概念
- 不知后世之事
- 保持历史准确性
- 使用符合时代的语言
